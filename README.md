# Lackadaisical

Lackadaisical (adjective), /ˌlæk.əˈdeɪ.zɪ.kəl/: showing little enthusiasm and effort. Syn.: half-hearted, apathetic, lethargic

This work by Nino Filiu is a piece of interactive media where low quality short form content melts right in front of the user's eyes.
